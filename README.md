# 📞 Chatbot Support System

이 프로젝트는 고객 지원을 위한 챗봇 기반 서비스로, Spring Boot를 기반으로 구축되었습니다.  
FAQ 답변, 사용자 요청 처리, 간단한 대화 로그 저장 기능을 제공합니다.

- **프로젝트명:** Chatbot Support System
